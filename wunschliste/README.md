# Foreverfuchs Hochzeit - No Name + Cooldown

## Änderungen

- Gäste müssen keinen Namen mehr eingeben
- Jeder Browser/jedes Gerät hat eigenen 10-Minuten-Timer
- Nach einem Wunsch sind Wunsch-Buttons 10 Minuten gesperrt
- DJ-Seite hat keine Status-Buttons mehr
- DJ-Seite zeigt nur Spotify/YouTube-Buttons

## Wichtig

Der 10-Minuten-Schutz läuft pro Browser über localStorage.
Das verhindert normales Spammen, ist aber kein perfekter Schutz gegen absichtliches Umgehen.

Für echten Missbrauchsschutz braucht man später eine Supabase Edge Function oder eigenes Backend.
